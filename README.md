![Demo Image](/demo.png)

## Installation

- Clone the repo
- `cd` into dir and run `yarn install`
- To run a dummy database, run `json-server --watch db.json --port 4000`
- To start the project, run `yarn start`
- Visit `http://localhost:3000`
